  this demo is based on the at-start board, in this demo, provides a basic
  communication between usart1 and usart2 in half-duplex mode. if received data
  right, the three leds will blink.

  set-up
  - connect usart2 tx pin (pa2) to usart1 tx pin (pb6)
  - connect a pull-up resistor to this line (10k).

  for more detailed information. please refer to the application note document AN0099.
