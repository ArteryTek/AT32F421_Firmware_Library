  this demo is based on the at-start board, in this demo, shows how to use
  halfduplex mode transfer data by interrupt mode.
  the pins connection as follow:
  - spi2 slaver        spi1 master
  - pb13     <--->     pa5
  - pb14     <--->     pa7
