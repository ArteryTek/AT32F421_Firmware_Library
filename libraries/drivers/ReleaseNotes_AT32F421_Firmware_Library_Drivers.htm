Release Notes for

AT32F421 Firmware Library Drivers

V2.0.7-2025/11/24

1.         ADC

l  更新adc_ordinary_conversion_trigger_set()函数，删除无效流程。

2.         CMP

l  更新cmp_filter_config()函数，设置FALSE时强制滤波值为0。

3.         ERTC

l  更新ertc_alarm_sub_second_get()函数写法，使用UNUSED标识处理AC6下的编译警告。

l  删除未使用的ERTC_OUTPUT_WAKEUP定义。

4.         GPIO

l  新增gpio_bits_toggle()函数。

5.         TMR

l  更新tmr_pwm_input_config()、tmr_encoder_mode_config()和tmr_output_channel_config()函数。通道模式配置前关闭通道使能，否则通道模式无法切换。

6.         更新部分注释。

1.         更新部分注释。

 

V2.0.5-2024/08/08

1.         更新i2c_flag_clear和i2c_init函数。

2.         更新部分注释。

4.         更新IRTMR驱动。

4.         更新部分注释。
